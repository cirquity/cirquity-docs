Using Cirquity

Setting up a new wallet

There are multiple types of wallets you can use with Cirquity.
To view a list of them, their interface and a brief description, as a well as guides on how to use them, you can check this guide.

Start mining Cirquity

Mining is essentially the process of using your computer to help verify and secure a cryptocurrency network, in our case, it is the Cirquity network. By doing this, you will be rewarded with CIRQ. If you want to learn about cryptocurrencies, mining is a great place to start!

To view an in-depth guide on how to mine Cirquity, you can view this guide
